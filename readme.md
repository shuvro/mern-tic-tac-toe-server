# Tic Tac Toe Server by Node.js

Client Server at https://github.com/shuvro/mern-tic-tac-toe-client.git

### Start

````bash
npm install
````

````bash
npm run build
````

### Development

```bash
npm run dev
```

### Running tests

```bash
npm test
```

### Docker
Run the Game Backend (Node.js) using docker

```bash
docker-compose up --build
```
Checkout the application at 
```bash
http://localhost:3000
```
